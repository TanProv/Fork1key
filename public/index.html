Câu hỏi thường gặp (FAQ)

Tại sao tôi cần xác minh?

Xác minh giúp nâng cấp tài khoản của bạn lên Google AI Studio Pro/Enterprise để sử dụng Gemini 1.5 Pro với giới hạn cao hơn.

Tôi có thể sử dụng bao nhiêu ID cùng lúc?

Hệ thống cho phép bạn nhập nhiều ID cùng lúc (mỗi dòng một ID) để xác minh hàng loạt, giúp tiết kiệm thời gian.

API Key lấy ở đâu?

Vui lòng liên hệ Admin để nhận mã API Key hợp lệ cho việc sử dụng công cụ xác minh này.

Nếu xác minh thất bại?

Hệ thống có cơ chế "Smart Refund" - nếu xác minh thất bại, lượt quota của bạn sẽ được hoàn lại tự động.
